amSTATZ Application Styleguide

This is the style guide for the application portion of amSTATZ.

We are using the Foundation 5, the latest version. You can always refer to their docs if you have detailed questions regarding a specific style of the framework.

Logo

These are the various ways the amSTATZ logo can be used. The logo color should never be any other color than white (#FFF) or blue (#1E96BC). The logo should never have a white background unless being used on a white page. Download logos here.
